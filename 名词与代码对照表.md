# 智能面试Agent系统名词与代码对照表

本文档用于维护论文中使用的核心名词、流程阶段与实际代码实现的对应关系，确保论文描述与工程实现的一致性。

## 1. 核心Agent与模块对照

| 论文名词 | 代码实现/类名 | 所在文件路径 | 备注 |
| :--- | :--- | :--- | :--- |
| Coordinator (协调器) | `InterviewBot` / `StageRouter` | `backend/agent/interview_bot.py` | 负责统筹面试流程和路由 |
| QuestionGenerator (问题生成器) | `BaseAgent` 的各个子类 (如 `OpeningAgent`, `CodingAgent` 等) | `backend/agent/agents/` | 负责生成具体阶段的面试问题 |
| Evaluator (评估器) | `EvaluationAgent` | `backend/agent/evaluation_agent.py` | 负责实时打分和生成评估报告 |
| InsightExtractor (洞察提取器) | `ResumeAnalyzer` | `backend/services/resume_analyzer.py` | 负责解析简历并提取候选人特征 |

## 2. 面试状态机阶段对照

## 4. 数据库核心表结构对照
