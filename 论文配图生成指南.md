# 毕设论文配图生成指南与提示词记录 (Gemini 生图版)

本文档整理了《基于MindSpore的面试Agent智能体》论文中所有需要绘制的图表，并提供了专门用于喂给 **Gemini 图像生成模型 (或 Midjourney/DALL-E 3)** 的自然语言提示词（Prompt）。

> **💡 提示**：AI 图像生成模型（如 Gemini Imagen）非常擅长生成具有科技感、学术风的高质量概念图和UI原型，但可能无法完美拼写图表中的具体中文文字。建议使用这些提示词生成高质量的底图后，再使用 PPT、Figma 或 Photoshop 加上准确的中文标签。

## 二、 数据图表 (带具体数值的精准提示词)

这类提示词用于生成高质量的学术数据图表底图。
**⚠️ 核心技巧**：为了避免生图模型画出“假大空”的通用图表模板，**必须在提示词中提供具体的坐标轴标签、数值范围、甚至具体的关键数据点**。这样生成的图表才会看起来像真实的实验数据。

### 1. [图4.2] 四维量化评估雷达图示例
**Gemini 生图提示词**：
> A professional data visualization radar chart (spider chart) with four axes labeled: 'Technical Depth', 'Communication', 'Problem Solving', and 'Code Quality'. The scale goes from 0 to 100. A translucent blue polygon represents the candidate's score, with data points explicitly plotted at approximately 85 for Technical Depth, 70 for Communication, 90 for Problem Solving, and 80 for Code Quality. Clean academic style, minimalist, scientific publication quality, white background, vector illustration.

### 2. [图5.1] 全本地化架构各阶段耗时占比图
**Gemini 生图提示词**：
> A professional stacked bar chart comparing two systems: 'Cloud API' and 'Local MindSpore'. The Y-axis represents 'Latency (ms)' ranging from 0 to 2000. Each bar has three color-coded segments: 'Network I/O' (bottom), 'Model Inference' (middle), and 'Data Processing' (top). The 'Cloud API' bar totals around 1500ms (Network 800ms, Inference 500ms, Processing 200ms). The 'Local MindSpore' bar totals around 600ms (Network 50ms, Inference 400ms, Processing 150ms). Clean academic data visualization style, scientific publication quality, white background.

### 3. [图5.2] 系统评分与专家评分散点拟合图

> A professional scatter plot data visualization showing the correlation between 'Expert Score' (X-axis, range 60-100) and 'AI System Score' (Y-axis, range 60-100). There are about 50 blue data points tightly clustered around a diagonal red trend line (y = x). Specific data points are visible at (85, 84), (92, 90), and (78, 80), showing a high Pearson correlation (r=0.92) annotated on the chart. Clean academic style, scientific publication quality, grid lines, white background.

### 4. [图5.3] 上下文压缩算法Token消耗对比折线图
**Gemini 生图提示词**：
> A professional line chart comparing two data series over 'Interview Turns' (X-axis, 1 to 20). The Y-axis is 'Token Count' ranging from 0 to 32000. Clean academic data visualization, scientific publication quality, grid background, white background.
